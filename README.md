**URL Params**

Keycloak kann farblich ueber URL-Parameter angepasst werden:

- `accent`: Akzentfarbe als Hex-Wert, mit oder ohne URL-kodiertes `#`, z. B. `accent=%23f2c45b` oder `accent=f2c45b`.
- `fontColor`: Textfarbe auf der Akzentfarbe, erlaubt sind `white` oder `black`.
- `lang`: Sprache der Login-Seiten, erlaubt sind `german` oder `english`.
- `kc_locale` / `ui_locales`: Locale-Hints fuer Keycloak/OIDC, z. B. `de` oder `en`.
- `hl`: Google-Language-Hint, z. B. `de` oder `en`. Muss in Keycloak beim Google-Provider als forwarded query parameter erlaubt sein.
